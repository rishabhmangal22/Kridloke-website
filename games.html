Stamina Run

Game Design

Stamina run is a 3d run game in which the player and the opponent chasing the player
both have stamina and the player has to outrun the opponent to win.

Prototype

This is the prototype of stamina run. Its a basic level which has a running track and a playable character mousy who has
Snamina,can run, change direction and jump and stops running if touches the boundry or the obstacle. The character has a cinematic camera also.
